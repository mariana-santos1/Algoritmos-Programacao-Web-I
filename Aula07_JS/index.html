Atividade JS

1.Como aconteceu a evolução da tecnologia JavaScript no decorrer dos anos?

O JavaScript evoluiu de uma linguagem simples de script para navegadores em 1995 para uma das tecnologias mais versáteis do mundo do desenvolvimento. Criado em apenas 10 dias por Brendan Eich na Netscape, seu objetivo inicial era apenas adicionar interatividade básica às páginas web.

  • Fase Inicial (1995-2005): Após disputas de navegadores, o JavaScript foi padronizado como ECMAScript para garantir funcionamento consistente.
  • Revolução Ajax (2005): A popularização do AJAX permitiu atualizações de conteúdo sem recarregar a página, tornando a web dinâmica e impulsionando o surgimento de bibliotecas como jQuery.
  • Era Moderna e Servidor (2009-Presente): O lançamento do Node.js em 2009 permitiu o uso do JavaScript no back-end (servidor), unificando o desenvolvimento da aplicação.
  • Padronização ES6+ (2015+): A atualização ECMAScript 2015 (ES6) trouxe recursos modernos, classes e módulos, tornando o código mais organizado. Frameworks como React, Angular e Vue consolidaram o JS na criação de SPAs (Single Page Applications) robustas.

    • 2.Qual a diferença entre Java e JavaScript?

    A linguagem de programação JavaScript, desenvolvida pela Netscape, Inc., não faz parte da plataforma Java. JavaScript não cria applets ou aplicativos independentes. Em sua forma mais comum, o JavaScript reside dentro de documentos HTML e pode fornecer níveis de interatividade às páginas da web que não são alcançáveis com o HTML simples. As principais diferenças entre Java e JavaScript são que o Java é uma linguagem de programação orientada a objetos, enquanto JavaScript é uma linguagem de script orientada a objetos. O Java cria aplicações que são executadas em uma máquina virtual ou navegador, enquanto o código JavaScript é executado apenas em um navegador. O código Java precisa ser compilado, enquanto o código JavaScript é todo em texto. E eles também requerem plug-ins diferentes.

    3.Quais as formas de se trabalhar com JavaScript junto com html. Dê exemplos.

    Existem três formas principais de integrar JavaScript (JS) com HTML: externo, interno e inline. A melhor prática, especialmente para projetos reais, é o uso de arquivos externos, pois facilita a organização e a manutenção. Externo envolve criar um arquivo separado com a extensão .js e vinculá-lo ao documento HTML usando a tag script com o atributo src. No interno, o código JavaScript é escrito diretamente no documento HTML, dentro da tag script, sem o atributo scr. E no inline o código JavaScript é inserido diretamente dentro das tags HTML através de atributos de eventos, como onclick, onmouseover, onload, entre outros.

    4.Como funciona o processamento do JavaScript em uma página web?

    O processamento de JavaScript (JS) em uma página web é realizado pelo navegador do cliente (client-side) através de um "motor" (engine) de JavaScript embutido, como o V8 (Chrome/Edge), SpiderMonkey (Firefox) ou JavaScriptCore (Safari). O processo transforma o código fonte em ações visíveis na tela seguindo processos como Parsing (Análise Sintática), Compilação JIT (Just-In-Time), Execução e Manipulação do DOM (Event Loop).

    5.Qual a importância do JavaScript em um projeto Web?

    O Javascript transforma páginas estáticas (HTML/CSS) em experiências interativas e dinâmicas. Ele executa scripts no navegador do usuário (client-side), permitindo atualizações de conteúdo em tempo real sem recarregar a página, além de validar formulários e controlar multimídias, melhorando significativamente a UX. Sem o JavaScript, a web seria limitada a documentos estáticos de leitura.

    6.Pesquise e cite quais frameworks para JavaScript existem atualmente no mercado.

    Os principais frameworks e bibliotecas JavaScript no mercado em 2026 incluem React, Angular, Vue.js, Next.js, Svelte/SvelteKit, SolidJS, Qwik, Astro, Remix, NestJS e Ember.js. A escolha está se tornando mais voltada para o problema específico (ex: SEO pesado = Astro/Next; Alta interatividade = Solid/React) do que apenas seguir o framework mais popular.
